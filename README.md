# Тестовое задание в компанию Ecom.tech
[Ссылка на pages](https://mestr3z.github.io/ecom.tech-test/)

## Реализовано
- Отображение товаров в виде карточке
- Поиск товаров 
- Модальное окно с подробной информацией о товаре
- Закрытие модалки разными вариантами

## Использованные технологии
- React
- TS
- Webpack
- CSS

## Установка и запуск

```bash
npm install

npm start

npm run build

```
